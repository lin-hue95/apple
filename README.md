                    <font face="Noto Serif HK Black" color="#eaeef7" size="3">
                      午夜書屋藏在巷弄深處，只在夜裡亮起招牌。進門先聞到紙張與木頭的氣味，
                      書架以小說為主，搭配少量散文、短篇與選讀。店內音量很低，
                      翻頁聲會比音樂更清楚。你可以慢慢挑書、慢慢坐下、慢慢讀完一段心情。
                      <br><br>
                      這裡的每一盞燈都故意留得不亮，因為我們相信：深夜最需要的不是更多刺激，
                      而是一個能讓思緒停下來的空間。
